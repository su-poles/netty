一、通道和流：
1. 通道可以同时进行读写，流只能读或者只能写, 比如FileInputStream或者FileOutStream，只能读或者写
2. 通道可以实现异步读写数据
3. 通道可以从缓冲区读数据，也可以写数据到缓冲区

channel --> Buffer
channel <-- Buffer

二、Channel是一个接口，其下面有子接口和实现类
常用Channel: FileChannel、DatagramChannel、ServerSocketChannel、SocketChannel


三、FileChannel主要对用来对本地文件进行IO操作，常见的方法有：
        （注意读写方向，记住一点：操作对象肯定是Channel， 读，那就是从通道读，写那就是写入通道，拷贝或者写入全都是操作的通道，既然是FileChannel，那操作对象肯定是个Channel）
    （1）public int read(ByteBuffer dst)     从通道读取数据并写入缓冲区
    （2）public int write(ByteBuffer src)     从缓冲区中写入到通道里
    （3）public long transferFrom(ReadableByteChannel src, long position, long count)  从源通道复制数据到当前通道
    （4）public long transferTo(long position, long count, WritableByteChannel target) 把数据从当前通道写入目标通道（使用了零拷贝技术）
